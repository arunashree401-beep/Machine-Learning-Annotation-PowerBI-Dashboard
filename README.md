# 📊 Machine Learning Data Annotation Performance Dashboard

## Overview

This project showcases an interactive Power BI dashboard built using real operational data from a Machine Learning Data Annotation workflow.

The dashboard provides insights into productivity, annotation performance, task completion, and operational efficiency through interactive KPIs and visualizations.

## Dashboard Preview

---

## Key Metrics

- Total Hours Worked
- Total Frames Processed
- Total Labels Annotated
- Completion Rate
- Daily Hours Trend
- Productive Hours
- Average SPL (Annotation Speed)
- Task Status
- Interactive Date Filter
